Cascata, Especificidade e Combinadores

Regra da Cascata:

Cascading style sheets, ou Cascata, é uma regra que diz que no CSS sempre é lido de cima para baixo.

  • Ver os Displays
  • Ver propriedades

Especificidade:

Especificidade são os seletores específicos que possuem mais "pontos" do que os menos específicos (classe específica).

Diferença entre a Especificidade do seletor pela Tag, seletor pela Classe, e seletor pelo Id:

  1. O Id é o mais específico deles, então ele prevalece sobre a classe.
  2. Já a classe prevalece sobre o seletor pelo elemento (Tag).
  3. É a Tag é a última deles.

Combinação de seletores:

  • Primeiro h1: Serve para colocar dois seletores em uma mesma linha.
  • Primeira > h1: Seleciona apenas o h1 que é "filho da Div".
  • Primeira h1 + h1: Estiliza somente os h1 que antes deles tenham um h1.

Funcionamento das pseudo-classes:

  1. First-child: Estliza apenas o primeiro elemento.
  2. Last-child: Estliza o último elemento.
  3. Nth-child: Seleciona o número do "filho" para estilizar.
  4. Nth-of-type: Mesma função do Nth-child.

Diferenças entre Display-inline e Display block:

Display-inline = Só ocupa o espaço que precisa para existir (ficam um do lado do outro), não dá para mexer na altura e nem largura, e nos outros elementos, só dá para mudar na horizontal.

propriedades

  • Padding;
  • Margin;
  • Border;

Display block = Ocupa a largura na tela inteira, pode mexer em todos elementos.

propriedades

  • Width;
  • Height;
  • Padding
  • Margin
  • Border

Display-inline block: Um pouco dos dois.

Serventia das propriedades:

Margin -> Colocar espaço entre os elmentos do HTML e CSS.

Padding -> Colocar espaço entre a borda e o conteúdo.

Boder -> Colocar espaço fora dos elementos.

Shorthand:

  1. Coloca uma quantidade de Pixels no Padding.
  2. Adiciona outro Pixel, o primeiro fica em cima e embaixo, e esse segundo fica nos lados.
  3. Adiciona outro Pixel, substituindo o que está embaixo.
  4. Adiciona mais um Pixel, e esse substitui o que está no lado esquedo.

Sufixos:

  • Block = Padding apenas em cima e embaixo;
  • Inline = Padding apenas nas laterias;
  • Top = Padding só em cima;
  • Bottom = Padding só embaixo;
  • Left = Padding na esquerda;
  • Right = Padding na direita;

Ultilidade da propriedade Boz-sizing : border-box:

Essa propriedade permite colocar a largura e a altura que você definiu.
